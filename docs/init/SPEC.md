# アンケートシステム 要件定義書

最終更新: 2026-06-26

---

## システム概要

マルチテナント対応のアンケート管理システム。
管理者がアンケートを作成・管理し、参加者が回答する。

**技術スタック**
- バックエンド: Laravel 12
- フロントエンド: Livewire 3 + Alpine.js
- DB: MySQL
- キャッシュ/セッション: Redis
- インフラ: Docker

---

## ロール構成

| ロール | URL プレフィックス | 説明 |
|--------|-------------------|------|
| スーパー管理者 | `/super` | 組織・管理者アカウントを管理するシステム全体の管理者 |
| 管理者 | `/admin` | 自組織のアンケート・参加者・お知らせを管理 |
| 参加者 | `/app` | 自組織のアンケートに回答、お知らせを閲覧 |

---

## スーパー管理者機能（`/super`）

### ログイン
- メールアドレス＋パスワードで認証

### 組織管理
- 組織の一覧・作成・編集・削除
- 組織削除は論理削除（データ保持）

### 管理者アカウント管理
- 管理者の一覧・作成・編集・削除（組織に紐付け）
- 有効/無効の切り替え

---

## 管理者機能（`/admin`）

### ログイン
- メールアドレス＋パスワードで認証（自組織スコープ）
- パスワードリセット: スコープ外

### 参加者管理
- 参加者の一覧・作成・編集・論理削除
- 削除済み参加者は「削除済み」として識別可能なまま回答データを保持
- メールアドレスは同一組織内で一意（他組織との重複は許容）
- 有効/無効の切り替え

### アンケート管理

**アンケートの構造**
```
アンケート（タイトル上限255文字）
├── 見出し（タイトル上限100文字）※任意・1階層のみ
│   └── 質問（質問文上限500文字）
└── 質問（見出しなしでも可）
```

**作成・編集ルール**
- アンケートは複数作成可能、同一タイトルも許容
- 見出しタイトルは同一アンケート内で重複不可
- 見出し・質問ともにドラッグ&ドロップで並び替え可能
- 質問は見出しをまたいで移動可能
- 見出しを削除した場合、中の質問は「見出しなし」として残る
- 回答方式は現在「1〜5の5段階ラジオボタン」のみ、将来拡張できる設計（answer_typesテーブルで管理）

**状態管理**

| 状態 | 説明 |
|------|------|
| 下書き | 作成・編集可能、参加者には非公開 |
| 公開 | 参加者が回答可能、編集不可・下書きへの差し戻し不可 |

- 公開するには1問以上の質問が必要
- 公開後に修正が必要な場合は削除して新規作成

**削除**
- 論理削除（回答データは保持）

### ダッシュボード
- アンケートごとの回答人数
- 質問ごとの回答平均値（棒グラフ）
- グラフ種別は将来拡張できる設計（Chart.js使用）

### お知らせ管理
- タイトル（上限255文字）＋本文（上限10,000文字）で作成
- 全参加者共通（ターゲティングなし）
- 公開/非公開の切り替え
- 同一タイトルは重複可能

---

## 参加者機能（`/app`）

### ログイン
- メールアドレス＋パスワードで認証（自組織スコープ）
- パスワードリセット: スコープ外

### お知らせ
- 公開中のお知らせを一覧・詳細閲覧

### アンケート回答
- 公開中のアンケートが一覧表示
- 複数のアンケートに回答可能
- 手動「途中保存」ボタンあり（続きから回答可能）
- 最終提出後は閲覧のみ（編集不可）

---

## データ制約一覧

| 項目 | ルール |
|------|--------|
| アンケートタイトル | 重複可、255文字以内 |
| 見出しタイトル | 同一アンケート内で重複不可、100文字以内 |
| 質問文 | 500文字以内 |
| お知らせタイトル | 重複可、255文字以内 |
| お知らせ本文 | 10,000文字以内 |
| 参加者・管理者名 | 100文字以内 |
| 参加者メール | 同一組織内で一意 |
| アンケート公開条件 | 1問以上必要 |
| アンケート状態遷移 | 下書き→公開のみ（公開→下書きは不可） |
| 公開後のアンケート編集 | 不可 |
| 削除方針 | 全て論理削除（回答データ・参加者情報は保持） |
| 並行編集の競合 | 後勝ち（競合制御はスコープ外） |

---

## スコープ外（今回実装しない）

- パスワードリセット（管理者・参加者ともに）
- お知らせのターゲティング（全参加者共通のみ）
- アンケート公開後の編集
- 並行編集の競合制御
